(async) init()

call all the functions after the DOM is loaded, initialize our indexedDB for user notes storage, get notes from storage or add notes to the page, and create our Edit and Save buttons for our note input page. We check if a note id exists, to indicate whether it is an existing or new note. We check whether the preview of the window url is set to true to represent if the user is in view mode or edit mode for a given note. We disable the save button
